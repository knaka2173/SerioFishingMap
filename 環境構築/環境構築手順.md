## SaaSの登録
- 使用するSaaS製品は以下（いずれもブラウザでのしようとすること）
  - Jira
  - Figma
  - Github

## インストールが必要なツール
  - RancherDesktop（DockerDesktop代用）
    - 実装で必要（DynamoDBローカル）
    - [公式サイト](https://rancherdesktop.io/)
  - A5M2
    - ER図作成で使用
    - [公式サイト](https://a5m2.mmatsubara.com/)
  - VSCode
    - 実装で必要（フロント・バック両方）
  - node.js
    - 実装で必要
    - バージョン:22
    - パッケージ管理はnpmを使用
    - [公式サイト](https://nodejs.org/ja/download)

## 任意でインストールするツール
  - Postman
    - [公式サイト](https://www.postman.com/jp/downloads/)

## 環境構築手順
1. GithubよりGitリポジトリをローカルPCにクローン
1. 以下ディレクトリをVSCodeで開く
  - SerioFishingMap/Frontend/serio_fishingmap
1. CLIで```npm install```を実行
1. CLIで```npm run dev```を実行
  - エラー出ずに画面が表示できれば完了
